# 🌍 GlobeTrekAdventures - Travel Journal Web Application

GlobeTrekAdventures is a dynamic, full-stack web application designed for travel enthusiasts to explore, document, and manage travel experiences across top destinations in Sri Lanka. It features interactive dashboards for both users and administrators to handle travel content and system data seamlessly.

## 🚀 Features

* **Interactive User Dashboard:** Browse popular destinations across Sri Lanka, including Ella, Mirissa, Nuwara Eliya, and Sigiriya.
* **Media-Rich Content:** Integrated dynamic video guides (`.mp4`) and high-quality image galleries tailored for each location.
* **Role-Based Admin & Staff Panels:** Complete backend control for system management, featuring separate Admin and Staff access levels.
* **Secure Authentication:** Dedicated, secure registration and login workflows for users, administrators, and staff members.
* **Dynamic Functionalities:** Interactive JavaScript scripting utilized for modern UI component rendering and structured array data management.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom variables, responsive grid/flexbox layout), JavaScript (Modern DOM manipulation & dynamic interaction).
* **Backend:** PHP (Role-based access controls, session handling, and backend routing).
* **Database:** MySQL (Relational database management handled via the phpMyAdmin interface).
* **Media:** Optimized streaming for local location videos and high-resolution image components.

## 📂 Project Structure

A quick overview of the core components within this repository:
* `index.php` - The main landing page welcoming visitors to the platform.
* `login.php` / `register_admin.php` - Secure user and administrative authentication modules.
* `userdashboard.php` - A personalized space for travelers to explore destinations.
* `admin_login.php` / `Administrator.php` - Secured dashboard endpoints for system administrators.
* `style.css` & `script.js` - Core UI styling sheets and relational interactive scripts.
